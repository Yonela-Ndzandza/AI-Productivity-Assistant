# GlowFlow AI

An AI-powered workplace productivity assistant designed to help women manage hormonal acne while staying productive at work. GlowFlow combines productivity tools with wellness support, offering AI-assisted meal planning, task scheduling, email drafting, and inflammation tracking.

## Project Overview

GlowFlow AI is a SaaS-style web application that bridges workplace productivity and wellness. It helps users plan gut-friendly, anti-inflammatory meals, track potential acne triggers, and leverage AI for common work tasks like email generation, meeting summaries, and research assistance. The app was built as a polished, modern web application suitable for presentation as a university capstone project.

## Features

- **Dashboard** – Central overview of productivity and wellness highlights.
- **AI Email Generator** – Generate professional email drafts based on context and tone.
- **Meeting Notes Summarizer** – Summarize meeting notes into actionable takeaways.
- **Task Planner** – Build a smart daily schedule with AI-suggested priorities.
- **Research Assistant** – Ask questions and get AI-curated research summaries.
- **AI Chatbot** – Conversational assistant for productivity and wellness guidance.
- **Meal Planner** – Generate a 7-day anti-inflammatory meal plan.
- **Grocery List** – Generate shopping lists based on meal plans.
- **Wellness Tracker** – Track food and lifestyle triggers that may affect skin health.
- **Settings** – Configure preferences and app behavior.

## Tools Used

- **TanStack Start** – Full-stack React 19 framework with SSR and server functions.
- **React 19** – UI library for building the component-based interface.
- **TypeScript** – Type-safe development across the entire stack.
- **Tailwind CSS v4** – Utility-first styling and custom design tokens.
- **Lovable AI Gateway** – Server-side AI integration for text generation and chat.
- **Vite 7** – Fast build tooling and development server.

## Setup Instructions

1. **Clone the repository**

   ```sh
   git clone <this-repository-url>
   cd <repository-name>
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Start the development server**

   ```sh
   npm run dev
   ```

4. **Open the app**

   Navigate to `http://localhost:8080` in your browser.
